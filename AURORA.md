# Фон hero — «Живая аврора»

Фон главного экрана — это анимированный меш-градиент: четыре размытых
цветных пятна медленно плавают на однотонной базе. Никаких картинок, чистый
CSS. Всё настраивается через переменные — **код трогать не нужно**, только цифры
и цвета.

## Где крутить

Файл: **`src/components/Hero.astro`**.
Найди блок `.hero__bg` (внутри `<style>`, ближе к началу). Там лежат все ручки:

```css
.hero__bg {
  --aurora-bg: #efe9f2;     /* база под пятнами */
  --aurora-c1: #b14cff;     /* фиолетовый */
  --aurora-c2: #3aa0ff;     /* синий */
  --aurora-c3: #ff3d6e;     /* розовый */
  --aurora-c4: #19d3b5;     /* бирюза */
  --aurora-speed: 1;        /* множитель скорости: >1 медленнее, <1 быстрее */
  --aurora-blur: 60px;      /* мягкость пятен */
  --aurora-size: 62%;       /* размер пятна */
}
```

Меняешь значение справа от `:` — сохраняешь — обновляешь страницу. Всё.

## Что какая ручка делает

| Переменная | За что отвечает | Примеры |
|---|---|---|
| `--aurora-bg` | Цвет фона-подложки под пятнами. Лучше держать светлым — на нём читается тёмный текст карточки. | `#efe9f2`, `#0f1016` (тёмная тема), `#fff` |
| `--aurora-c1…c4` | Четыре цвета пятен. Можно ставить любой CSS-цвет. | `#b14cff`, `hotpink`, `rgb(255 80 0)` |
| `--aurora-speed` | Скорость движения. `1` — как сейчас. `2` — вдвое медленнее/спокойнее. `0.5` — вдвое быстрее/живее. | `1`, `1.5`, `0.7` |
| `--aurora-blur` | Насколько пятна размыты. Больше = мягче и «туманнее», меньше = резче и контрастнее. | `60px`, `90px`, `40px` |
| `--aurora-size` | Размер пятен. Больше = заполняют сильнее, меньше = больше видно базу. | `62%`, `80%`, `50%` |

### Как подобрать цвета
Любой из `--aurora-c1…c4` можно задать:
- HEX: `#ff3d6e`
- по имени: `tomato`, `deepskyblue`, `gold`
- RGB: `rgb(255 61 110)`

Совет: бери 3–4 цвета из одной «температуры» (все тёплые или все холодные) — будет
гармоничнее. Полный разнобой смотри по вкусу.

## Тёмная тема (пример)

Хочешь как вариант «Неон-флюид» со свечением на тёмном — поставь тёмную базу и
сочные цвета:

```css
--aurora-bg: #0f1016;
--aurora-c1: #ff2fb0;
--aurora-c2: #6a5cff;
--aurora-c3: #18e0c8;
--aurora-c4: #ffc24b;
```

⚠️ На тёмной базе тёмный текст карточки потеряется. Если делаешь тёмную тему —
напиши разработчику, нужно будет осветлить текст карточки.

## Чуть глубже (по желанию)

- **Сколько пятен.** Сейчас их 4 (`.hero__blob--1…4` в разметке и в стилях).
  Можно убрать одно или добавить пятое, но это уже правка разметки — лучше с
  разработчиком.
- **Рандом.** Каждое открытие страницы аврора начинается с другой фазы и чуть
  другой скоростью — поэтому дважды одинаково не выглядит. Это делает маленький
  скрипт внизу файла, отдельно настраивать не нужно.
- **Бережно к глазам.** Если у пользователя в системе включено «уменьшить
  движение» (reduced motion), анимация автоматически выключается и аврора
  замирает красивой статичной картинкой. Так и задумано.
